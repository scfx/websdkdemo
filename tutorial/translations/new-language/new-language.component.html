How to add a new language

Add a new language

  • Create a new .po file with the Poedit program or manually. If you choose the second option, add the following metadata to it. 
    
          "Project-Id-Version: c8yui.core\n"
          "Report-Msgid-Bugs-To: \n"
          "POT-Creation-Date: \n"
          "PO-Revision-Date: \n"
          "Last-Translator: \n"
          "Language: it\n"
          "Language-Team: \n"
          "Content-Type: text/plain; charset=UTF-8\n"
          "Content-Transfer-Encoding: 8bit\n"
          "Plural-Forms: nplurals=2; plural=(n != 1);\n"
  • Add translations to this file: 
    
          msgid "Devices"
          msgstr "Dispositivi"
  • In the index.ts file, add the following line: import "./locales/it.po";
  • In the package.json file, under the c8y keys add: {{ lang | json }}
